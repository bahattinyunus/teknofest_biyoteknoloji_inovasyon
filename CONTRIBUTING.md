# Katkıda Bulunma Rehberi

BioSentez projesine katkıda bulunmak istediğiniz için teşekkür ederiz! Açık bilim ve açık kaynak felsefesini benimsiyoruz.

## Nasıl Katkıda Bulunabilirsiniz?

1. **Hata Bildirimi:** Projede gördüğünüz eksiklikleri veya hataları "Issues" sekmesinden bildirebilirsiniz.
2. **Özellik Önerisi:** Yeni fikirlerinizi tartışmaya açabilirsiniz.
3. **Pull Request:**
    *   Repo'yu fork edin.
    *   Yeni bir dal (branch) oluşturun: `git checkout -b feature/YeniOzelik`
    *   Değişikliklerinizi yapın ve commit edin: `git commit -m 'Özellik: X eklendi'`
    *   Dalınızı pushlayın: `git push origin feature/YeniOzelik`
    *   Bir Pull Request oluşturun.

## Kod/Veri Standartları

*   Biyolojik veriler (DNA sekansları) `.fasta` veya `.gb` formatında olmalıdır.
*   Simülasyon kodları için lütfen Python (PEP8) standartlarına uyunuz.
*   Akademik referanslar veriyorsanız lütfen DOI numaralarını ekleyiniz.
